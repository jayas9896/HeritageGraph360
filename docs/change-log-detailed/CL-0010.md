# CL-0010: Fix gRPC Generated annotation dependency

## Summary
This change adds the javax.annotation API dependency required by generated gRPC sources.

## Motivation
gRPC generated code references javax.annotation.Generated which is not on the classpath by default in Java 17.

## Changes
- Added javax.annotation-api dependency to ingestion-service.

## Verification
- mvn -q -DskipTests=false verify (pass).

## Risks
- Low risk; adds a compile-time dependency for generated sources.

## Rollback
- Remove the javax.annotation-api dependency from ingestion-service.

## Notes
- Consider updating gRPC generation to use jakarta annotations in future.
