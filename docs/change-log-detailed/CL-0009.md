# CL-0009: Add Shibboleth repository for SAML dependencies

## Summary
This change adds the Shibboleth releases repository required to resolve OpenSAML artifacts for SAML support.

## Motivation
Spring Security SAML transitively depends on OpenSAML artifacts that are not available in Maven Central.

## Changes
- Added Shibboleth releases repository to the parent POM.

## Verification
- Pending: `mvn -q -DskipTests=false verify`.

## Risks
- Low risk; adds an additional Maven repository for build resolution.

## Rollback
- Remove the repository entry from the parent POM.

## Notes
- Ensure repository availability in CI environments.
