# CL-0022: Add RBAC provisioning APIs and identity admin security

## Summary
This change adds RBAC provisioning endpoints and secures them with JWT-based admin access.

## Motivation
Role and permission management require explicit admin APIs with strong access controls.

## Changes
- Added RBAC provisioning service and controller endpoints.
- Added identity admin security configuration for RBAC APIs.
- Added OAuth2 resource server dependency to identity-service.
- Updated REST API documentation.

## Verification
- mvn -q -DskipTests=false verify (pass).

## Risks
- RBAC APIs rely on JWT roles being mapped correctly.

## Rollback
- Remove RBAC APIs and admin security config.

## Notes
- Align JWT claims with identity provider roles.
