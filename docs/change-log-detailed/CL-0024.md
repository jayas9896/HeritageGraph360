# CL-0024: Add policy management APIs

## Summary
This change adds policy management endpoints for admin users.

## Motivation
Policy provisioning is required for fine-grained access control.

## Changes
- Added policy controller and request/response DTOs.
- Secured policy endpoints in identity admin security config.
- Updated REST API documentation.

## Verification
- Pending: `mvn -q -DskipTests=false verify`.

## Risks
- Policy definitions are stored without validation of conditions JSON.

## Rollback
- Remove policy controller and DTOs and revert security config.

## Notes
- Consider adding schema validation for policy conditions.
