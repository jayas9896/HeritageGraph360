# CL-0007: Fix Maven parent resolution for module builds

## Summary
This change updates module POMs to resolve the parent POM via relative path for local builds.

## Motivation
The multi-module build must resolve the parent POM when building from the repo root.

## Changes
- Added parent relativePath entries in all module POMs.

## Verification
- mvn -q -DskipTests=false verify (pass).

## Risks
- Low risk; affects only Maven resolution.

## Rollback
- Remove the relativePath entries from module POMs.

## Notes
- Keep parent POM at repository root.
