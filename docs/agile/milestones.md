# Milestones

1) M1 - Foundation: Maven modules, base docs, UML, CI checks.
2) M2 - Security: OAuth2, SAML, RBAC, gateway routing.
3) M3 - Core Domain: profiles, relationships, audit logs.
4) M4 - Ingestion: gRPC, Kafka, Kinesis, NoSQL event store.
5) M5 - Insights: RabbitMQ jobs, anomaly scoring, reports.
6) M6 - Observability: Prometheus, Grafana dashboards, runbook.
7) M7 - Deployment: Docker and local Kubernetes.
